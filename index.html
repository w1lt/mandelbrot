About

The Mandelbrot set is the set of complex numbers c for which the function f(z) = z^2 + c does not diverge when iterated from z = 0. This means that, given a complex number c, we can plug it into the function f(z) and get a new complex number. We can then plug that number back into the function and get another number, and so on. If the sequence of numbers we get from this process stays bounded, then c is in the Mandelbrot set. If the sequence diverges, then c is not in the Mandelbrot set. Here is a great video that explains the Mandelbrot set in more detail.

This visualizer allows you to explore the Mandelbrot set by changing the base color, max iterations, resolution, and zoom level. You can also randomize the colors to get a different look each time.
